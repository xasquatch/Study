box-sizing

요소의 실제 크기는 width속성과 height속성에 padding값과 border값이 더해져서 브라우저에 나타난다.
(box-sizing:content-box;이 기본값임)
따라서 padding을 추가하거나 border를 두껍게 주면 레이아웃이 계획과 달리 두줄로 밀려날 수도 있다.
이때 box-sizing속성을 border-box로 지정하면 padding값과 border값이 width와 height에 포함 되어서
요소의 실제 크기가 더이상 커지지 않는다.

box-sizing: content-box;? 요소의 크기=content
box-sizing: border-box;? 요소의 크기=content+padding+border

box-sizing: content-box;
box-sizing: border-box;
