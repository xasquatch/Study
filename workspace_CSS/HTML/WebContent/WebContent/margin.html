margin : 바깥 여백

padding수업 보기

요소의 테두리(border)를 기준으로 바깥에 생기는 4방향의 여백을 margin이라 함. 요소의 위치에 변화가 생긴다.

block요소 자체를 수평으로 항상 가운데 정렬하려면 좌우 margin의 값을 auto로 지정한다.

margin-(top/right/bottom/left)

  1. margin-top : 위쪽 바깥 여백
  2. margin-right : 오른쪽 바깥 여백
  3. margin-bottom : 아래쪽 바깥 여백
  4. margin-left : 왼쪽 바깥 여백

inline요소에서의 margin 적용

inline요소 즉, 줄이 바뀌지 않는 요소들은 margin이 상하 방향은 적용되지 않고 좌우는 적용된다.

span-1 span-2 span-3 span-4 b

margin:n/n n/n n n n

margin:100px;

margin:0 100px;

margin:100px 0;

margin:0 100px 150px 200px;
