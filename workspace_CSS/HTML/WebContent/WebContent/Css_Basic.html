CSS Basic

CSS 적용 방식

  1. inline 방식 - 태그 뒤에 직접 적용.(적용되는 우선순위는 가장 높지만 권장하지는 않음)
    <tag style="속성:속성값;">
  2. embeded 방식 - 웹문서의 head영역내에 style태그를 써서 적용(해당 웹문서에만 적용됨.)
  3. external 방식 - 웹문서와 별개의 외부파일로 스타일을 정의한 파일명.css파일을 만들고 연결시켜서 적용
    (여러 웹문서에 공통된 스타일을 적용하고 수정하기 용이. 일관성 유지에 편리하다)
    - head영역 내에 link태그를 써서 적용 -> link href="파일명.css" rel="stylesheet"

CSS선택자 중의 기본 3가지

  1. 전체 선택자 -*
  2. ID 선택자 - 만들때는 id="id명", 불러올 때는 #id
  3. Class 선택자 - 만들때는 class="class명" 불러올 때는.class명
  4. Tag 선택자 - 불러올 때는 tag 그대로
    - 강제적이고 자동적으로 변함.
