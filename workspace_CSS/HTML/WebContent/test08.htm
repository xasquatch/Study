Image 지정하기

<img src="이미지경로" alt="이미지 설명글(이미지 대체텍스트)">
width와 height 속성으로 각각 브라우저에 나타낼 이미지의 가로 세로 크기를 지정.

WEB에서 자주 쓰이는 대표적인 이미지 형식

  • *.gif -
    256가지 색상 표현.
    투명배경 적용 가능. 움직이는 이미지(움짤) 가능.
  • *.jpg -
    트루색상 표현.
    일반적인(사실적인) 사진. 색상에 비해 파일 크기 적음
  • *.png -
    gif와 jpg의 장점을 갖춤.
    트루색상 표현, 색상에 비해 파일 크기 적음
    투명배경 적용 가능.
