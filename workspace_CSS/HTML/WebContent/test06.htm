form으로 사용자 입력 페이지 만들기

입력내용을 form으로 페이지 구성하고
form에 method속성과 action속성을 지정한다.
method의 값은 POST/GET중에서 선택하고
action의 값은 내용을 전달받을 서버의 페이지를 지정한다.

<label for=id와 동일값>input의 설명글내용</label>
<input type=text id=for와 동일값>
  1. 한줄 글상자 input type=text
  2. 암호 글상자 input type=password
  3. 객관식 고르기(한개만) input type=radio
  4. 객관식 고르기(두개 이상 가능) input type=checkbox
  5. 전송버튼 input type=submit
  6. 취소버튼 input type=reset
  7. 일반버튼 input type=button
  8. 버튼 button ... /button(기본은 전송상태. type 속성으로 역할을 지정)
