html5에서 추가된 input type의 값들

  1. time - 사용자의 지역을 기준으로 시간(시,분,초)을 지정
  2. date - 사용자의 지역을 기준으로 날짜(연,월,일)을 지정
  3. datetime-local - 사용자가 위치한 곳을 기준으로 설정된 날짜와 시간(연,월,일,시,분,초)을 지정
  4. month - 사용자의 지역을 기준으로 날짜(연,월)을 지정
  5. week - 사용자의 지역을 기준으로 날짜(연,주)을 지정
  6. url - URL주소를 포함하는 영역에 사용(일부 스마트폰에서는 유형을 인식하고 키보드에 .com을 추가함, 브라우저 자체에서 해당 형식에 맞는지를 검사)
  7. email - email주소를 포함하는 영역에 사용(일부 스마트폰에서는 유형을 인식하고 키보드에 .com을 추가함, 브라우저 자체에서 해당 형식에 맞는지를 검사)
  8. tel - 전화번호를 포함하는 영역에 사용(일부 스마트폰에서는 유형을 인식하고 숫자 키보드로 변경)
  9. number - 숫자를 포함하는 영역에 사용
    • min=n - 최소값
    • max=n - 최대값
    • step=n - 간격
  10. range - 범위 내의 값을 포함하는 영역에 사용
    • min=n - 최소값
    • max=n - 최대값
    • step=n - 간격
  11. search -
  12. color -

input type의 속성들

  1. placeholder=... - 입력할 예시
  2. autocomplete=on/off - 자동완성기능을 제어
  3. required - 전송하기 전에 반드시 기입해야 함
  4. autofocus - 커서가 미리 위치함
  5. readonly - 읽기 전용으로 지정함
  6. disabled - 비활성화 영역으로 지정함
