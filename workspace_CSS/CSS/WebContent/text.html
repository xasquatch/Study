font-size:npx/nem/nrem/n%/... 글자 크기;

브라우저에 나타나는 본문의 글자 크기는 16px이 기본값임. px단위는 항상 크기가 해당 값으로 고정이 되고, em단위는 상위요소의 크기에 영향을 받는 배수 단위이며, rem단위는 최상위요소 크기의 영향만 받는 배수 단위.

font-family : 서체(글꼴)

서체(글꼴)는 크게 장식이 있는 serif서체와 장식이 없는 sans-serif서체로 나눌 수 있다. 서체명을 따옴표로 묶어 순서대로 나열하면 된다. 마지막에 serif나 sans-serif를 따옴표 없이 지정한다.

한글서체(네이버) 구글서체 https://fonts.google.com

font-family:"Georgia","Times New Roman","Times",serif; 서체(글꼴)

font-family:"Verdana","Arial",sans-serif; 서체(글꼴)

font-family:서체(글꼴)

google/ font-family: Dokdo 서체(글꼴)

google/ font-family: Gothic A1 서체(글꼴)

text- 로 시작하는 속성

  • text-align : 수평방향으로 글자 정렬(해당 요소의 영역 내에서)
    1. left; 기본값
    2. right; 오른쪽
    3. center; 가운데
    4. justify; 양끝 정렬
  • text-decoration : 글자의 줄 관련
  • text-transform : 영문의 대소문자 지정
  • text-indent : 들여쓰기
  • text-overflow : 영역을 벗어난 텍스트 설정
  • text-shadow : 글자의 그림자
