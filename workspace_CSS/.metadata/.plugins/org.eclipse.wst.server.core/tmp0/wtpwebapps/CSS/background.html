background

  • background-color:색상 - 배경 색상 지정
  • background-image:url(이미지 경로) - 배경 이미지 지정

background-image의 속성들

  • background-repeat (배경이미지 반복) : repeat/no-repeat/repeat-x/repeat-y; (기본값/한번만/가로로만 반복/세로로만 반복)
  • background-position (배경이미지 위치) : n n ; 으로 가로값과 세로값을 지정하거나 : left/center/right 으로 수평방향값을 지정하거나 top/center/bottom; 으로 수직방향값을 지정할 수 있다.
  • background-size (배경이미지 크기): contain/cover/n% n%/npx npx/...; 정해진 값으로 지정하거나 다양한 단위를 붙여 가로크기와 세로크기에 직접 값을 줄 수 있다.
  • background-attachment(배경 고정/스크롤):scroll/fixed/local;
    브라우저나 특정 영역에 생긴 스크롤바의 움직임에 배경이미지가 어떻게 움직일지를 지정.

background-repeat (배경이미지 반복) : repeat/no-repeat/repeat-x/repeat-y;

background-repeat : repeat; 기본값(양사방 반복)

background-repeat : repeat-x; 가로로만 반복

background-repeat : repeat-y; 세로로만 반복

background-repeat : no-repeat; 한번만 나옴

background-position (배경이미지 위치)

background-position:left center;

background-position:center center ;

background-position:right bottom;

background-position:100px 50px;;

background-position:80% 30%;

background-position:10em 5em;

background-size (배경이미지 크기)

background-size:contain;
그림위주(그림 하나가 온전히 보임,빈공간이 생기거나 반복될 수 있음)

background-size:cover;
공간위주(그림이 부분적으로 보이지 않을 수도 있지만 그림 하나로 공간 전체를 덮음)

background-size:100% 100%; 가로 세로 순서

background-size:300px 400px;
