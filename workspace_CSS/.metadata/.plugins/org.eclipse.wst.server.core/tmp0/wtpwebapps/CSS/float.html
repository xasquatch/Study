float:left/right; 부유하다

float을 적용하면 요소가 브라우저 위에 뜬 상태가 되어 자연스럽게 배치됨. (이미지와 문단의 글자를 둘러싸듯이 자연스럽게 배치함) block요소들을 나란히 배치할 때도 쓰임.(block요소는 한 줄에 하나만 옴) left값을 적용한 요소들끼리는 왼쪽부터 나란히 배치. right값을 적용한 요소들끼리는 오른쪽부터 나란히 배치.(소스상의 요소 순서와 바뀜) float을 적용하면 뒤에 오는 모든 요소가 함께 영향을 받음. 만약 더이상 float 영향을 주지 않으려면 반드시 clear속성을 적용해햐 함.(both) 자식요소들이 모두 float의 영향을 받으면 부모요소는 높이값이 사라져서 줄어듬. (부모요소의 형태를 유지하려면 높이값을 지정해야 함)
