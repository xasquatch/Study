CSS Basic

CSS 적용방식

  1. inline 방식 - 태그 뒤에 직접 적용.(스타일이 적용되는 우선순위는 가장높음.)
    <tag style="속성:속성값;">
  2. embeded 방식 - 웹문서의 head영역내에 style태그를 써서 적용(해당 웹문서에만 적용됨.)
  3. external 방식 - 웹문서와 별개의 외부파일로 스타일을 정의한 *.css파일을 만들고 연결시켜 적용
    (여러 웹문서에 공통된 스타일을 적용하고 수정하기 용이. 일관성 유지에 편리함)
    *.css파일을 불러올 때는 웹문서의 head영역내에 link태그를 써서 적용.
    <link href="*.css" rel="stylesheet">

CSS선택자 중의 기본 3가지

  1. 전체 선택자 - *
  2. ID 선택자 - 만들 때는 id="id명", 불러올 때는 #id명
    동일한 id명을 중복해서 쓸 수 없으므로 주로 구조적인 필요에 의해 쓰임.
  3. Class 선택자 - 만들 때는 class="class명", 불러올 때는 .class명
    중복해서 쓸 수 있으므로 주로 디자인적인 필요에 의해 쓰임.
  4. Tag 선택자 - 불러올 때는 tag 그대로.
    강제적이고 자동적으로 적용 됨.
