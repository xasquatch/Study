transition

  • transition-property: 효과를 줄 속성명. 전체 속성에 적용할 경우는 값을 all
  • transition-duration: 진행시간(단위는 초-s,밀리초-ms)
  • transition-delay: 지연시간(단위는 초-s,밀리초-ms)
  • transition-timing-function: 가속,감속 효과
  • transition:함축형 표현(속성명 진행시간 지연시간...의 순서)

transition-timing-function

  • transition-timing-function: ease; 기본값. 처음에는 천천히 시작해서 점차 빨라지다 마지막에는 천천히 끝남
  • transition-timing-function: ease-in; 시작을 느리게 함
  • transition-timing-function: ease-out; 마지막을 느리게 함
  • transition-timing-function: ease-in-out; 느리게 시작하고 느리게 끝냄
  • transition-timing-function: linear; 시작부터 끝까지 똑같은 속도로 진행
  • transition-timing-function: cubic-bezier(n,n,n,n); bezier 함수를 직접 정의해서 사용
