------------평가지시----------

- 제시한 UI화면을 분석하여 구현하고
  본인 성함의 폴더 안에 작성한 HTML파일, CSS파일, img폴더 제출

[실기문제 요구사항]

- css는 외부파일 형태로 구현
- 브라우저의 정가운데 위치하는 레이아웃 구현
- UI 요구사항과 UI 표준에 따라 설계된 화면의 구조를 해석하여 구현
- UI 요구사항과 UI 표준에 따라 설계된 메뉴의 구조를 해석하여 구현
- 주메뉴 롤오버시 효과 구현 (각 메뉴별 텍스트 색상 4가지 구현)
   기업 : #f00
   문화 : #0f0
   제품 : #00f
   고객 : #ff0
- 링크 텍스트에 롤오버시 밑줄 효과 구현
