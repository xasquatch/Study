HTML이란?
Hyper Text Markup Language의 약자.
웹문서를 만들어 주는 프로그래밍 언어를 말함.



명령어를 가리켜 tag태그라 하며
태그는 기본적으로 시작태그와 종료태그의 한 쌍으로 구성되고 각괄호 안에 들어있다.
웹문서는 확장자를 *.htm 이나 *.html 로 지정하여 저장한다.
인코딩은 UFT-8로(웹표준).

문단을 구성하는 태그는
p태그 이며 시작과 종료 한 쌍으로 묶어준다.
문단 내에서 줄을 바꾸고 싶으면
br태그를 붙여준다.
br태그는 단독 태그이므로 선언만 하면 되고 중복적용이 가능함.

hn태그(h1~h6) - 제목글자 태그 알아보기

두꺼운 글자체, bold체로 표현되며
별도의 단락을 구성한다. (위아래로 한 줄의 공백)
