데이터 딕셔너리

v$datafile	데이터 파일의 위치, 이름, 번호와 같은 정보를 가지고 있음.
v$controlfile	컨트롤 파일의 위치, 이름, 다중화와 같은 정보를 가지고 있음.
v$log		컨트롤 파일에 있는 리두로그 파일의 정보
v$logfile	리두 로그 파일의 그룹 번호, 위치, 이름과 같은 정보를 가지고 있음.
v$database	데이터베이스의 기본 정보를 담고 있음. DB 이름, archive log mode 여부
v$session	현재 세션에 대한 정보를 담고 있음. 
		데이터 충돌 시 블락킹 세션의 정보도 볼 수 있음.
dba_sys_privs	사용자의 시스템권한 정보를 가지고 있음.
dba_tab_privs	사용자의 오브젝트권한 정보를 가지고 있음.
